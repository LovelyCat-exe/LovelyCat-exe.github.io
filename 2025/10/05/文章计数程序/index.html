Count Tool

这是一个简单的 C++ 工具，用来统计字符串或文本文件中的字符 / 单词数量。
它支持中英文混合文本，并提供灵活的参数来控制统计方式。

功能特性

  • 字符统计模式（默认）

    • 可以选择是否统计空格 (includeSpace)
    • 可以选择是否统计换行 (includeLineBreak)
    • 可以选择忽略指定的字符或字符串列表（支持全角字符）

  • EnglishMode 模式

    • 如果文本 全是中文，返回 -1
    • 否则返回英文单词的数量（连续字母/数字视作一个单词）

  • 忽略指定字符功能

    • 参数 ignoreSpecificCharacters = true
    • 提供 SpecificCharacters 列表（vector<string>）
    • 支持跳过特定的中英文子串，例如 "一"、" " 等

注意事项

  • example.cpp已经预制好了content, 如果想要自行输入, 可以取消第十行的注释, 把第十一行取消注释, 就可以手动输入了. 要注意: 程序会读取整个stdin 的内容, 也就是说你不能直接使用终端进行输入, 可以使用例如VScode 插件OI runner来实现从输入框读取; 或者你可以改用文件读写.
  • EnglishMode 模式下遇到中文不会报错，只是跳过。如果整段文本都是中文，则返回 (size_t)-1。
  • 默认情况下 SpecificCharacters 是空的，不会忽略任何字符。
  • 可以只传一个string s, 其他默认false
